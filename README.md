# 👗 Fashion Store App

A modern Single Page Application built with Vue 3, TypeScript, and Tailwind CSS.

## 🚀 Tech Stack
- Vue 3 (Composition API)
- TypeScript
- Vite
- Tailwind CSS
- Pinia (State Management)
- Vue Router

## ✨ Features
- Browse fashion products from DummyJSON API
- Search products by name
- Filter products by category
- View product details
- Add products to cart
- Cart persists across page reloads (localStorage)
- User authentication simulation (JWT)
- Dark mode toggle
- Fully responsive (Mobile, Tablet, Desktop)

## 📦 Installation

```bash
npm install
```

## 🔧 Run Development Server

```bash
npm run dev
```
